## 🧠 核心技术 | Technology

MasterAI 基于**博弈论(GTO) + 深度学习 + 强化学习**，采用以下前沿技术：

| 技术 | 说明 |
| :--- | :--- |
| **CFR算法** | 虚拟遗憾最小化，计算纳什均衡策略 |
| **蒙特卡洛采样** | 大量MC采样逼近最优行动价值 |
| **神经网络估值** | 离线求解+在线连续重解 |
| **纳什均衡策略** | GTO最优策略，无法被利用 |

### 架构组成

- **离线组件**：求解随机扑克局面，训练神经网络预测每手牌的价值
- **在线组件**：使用连续重解算法，动态选择最优行动
